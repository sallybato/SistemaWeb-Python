<h1>TaskFlow:</h1>

#

Sistema de Gerenciamento de Tarefas Pessoais
O TaskFlow é uma aplicação web simples desenvolvida em Python (Flask) para gerenciar tarefas e compromissos pessoais, com foco em autenticação e controle de acesso individualizado. Ele permite que cada usuário crie uma conta, faça login e gerencie suas próprias tarefas.
